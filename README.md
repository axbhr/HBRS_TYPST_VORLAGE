# H-BRS Typst Vorlage (inoffiziell)

Dies ist eine **inoffizielle** [Typst][1] Vorlage für wissenschaftliche Arbeiten,
inspiriert von den [LaTeX][2] Vorgaben des Fachbereichs Informatik der
[Hochschule Bonn-Rhein-Sieg][3].

> ⚠️ **Wichtiger Hinweis:** Dies ist kein offizielles Projekt der Hochschule
> und steht in keiner Verbindung zum Fachbereich oder dessen IT‑Angeboten.
> Die Vorlage wurde von Studierenden als Anpassung der LaTeX‑Vorlage erstellt.

## Einstieg

Um diese Vorlage nutzen zu können, muss [Typst][1] auf Deinem Rechner installiert sein. Eine Anleitung zur Installation findest du auf der
offiziellen Typst-Website.

## FAQ

### Schriftartenprobleme

Typst bringt eine eigene Font‑Verwaltung mit. Sollten bestimmte Schriftarten
fehlen, installiere sie über dein Betriebssystem und führe ggf. einen Neustart
der Typst-Engine aus. Es gibt keine äquivalenten Befehle zu `luaotfload` in Typst.
